Gokan

A tool to use the various environmental sensors available on common hardware running Linux.

Most "ultrabook" laptops have 3D gyroscopes, accelerometers, magnetometers, lid position sensors, ambient light detectors, and so on. None of these sensors are made available to userspace in any useful way. What a waste! Imagine what you could do if only your laptop knew where it was and where it was going!

Using Gokan

The Gokan project is still in its early days and does not yet have anythingh useful you can install and use.
